# Job Application Assistant - User Guide

Welcome to the Job Application Assistant! This guide will help you use the system effectively.

---

## Features

- Search for jobs across multiple platforms.
- Upload your resume for easy application.
- Apply to jobs directly from the dashboard.
- Manage your profile and job preferences.

---

## Getting Started

### Accessing the Application

1. Open your web browser.
2. Navigate to the frontend URL, typically:
   ```
   http://localhost:3000
   ```

---

## Using the Dashboard

### Viewing Job Listings

- The dashboard displays a list of jobs based on default search criteria.
- Each job shows the title, company, location, and a link to the job posting.

### Applying to a Job

- Click the **Apply** button on a job listing.
- You will receive a confirmation alert once the application is submitted.

### Uploading Your Resume

- Use the file input to select your resume file (PDF or DOCX recommended).
- Click the **Upload Resume** button to upload.
- A confirmation alert will notify you of successful upload.

---

## Profile Management

- Currently, profile creation and management are handled via backend APIs.
- Future frontend UI for profile management may be added.

---

## Troubleshooting

- If job listings do not appear, ensure the backend server is running.
- If uploads fail, check file format and size.
- For any errors, check the browser console and backend logs.
